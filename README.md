# CHONGI Global Website

Transforming Lives in Uganda through Community Empowerment

## 🌟 About

CHONGI Global is a non-profit organization dedicated to empowering youth and communities in Uganda through ICT training, education support, and livelihood development programs.

## 🚀 Deployment

### Quick Deploy

**Netlify (Recommended)**
1. Push to GitHub repository
2. Connect repository to Netlify
3. Deploy automatically

**Vercel**
1. Push to GitHub repository  
2. Import project in Vercel
3. Deploy automatically

**GitHub Pages**
1. Push to GitHub repository
2. Go to Settings > Pages
3. Select main branch as source

### Manual Deploy

1. Clone repository
2. Run `python -m http.server 8080` for local testing
3. Upload files to hosting provider
